## 🌷 Sobre

Este repositório funciona como um notebook técnico pessoal onde documento meus estudos, anotações, testes, conceitos e aprendizados relacionados à Cloud Computing.

A ideia é registrar minha evolução enquanto estudo diferentes tecnologias, serviços e fundamentos da área, transformando tudo em uma base de conhecimento pública e organizada.

Os conteúdos podem variar entre:

- anotações rápidas
- resumos
- troubleshooting
- snippets
- comandos úteis
- estudos teóricos
- testes práticos
- documentações pessoais
